# Wananchi Group Stores Requisition System (WGR-SRS)

A modern, role-based web prototype for a fully digital **Stores Requisition Process** — replacing paper forms with an intuitive, colour-coded workflow.

![Wananchi Group Logo](path-to-logo.png)

## 🎯 Overview

This is a **static HTML/CSS/JS prototype** demonstrating the future UX/UI of the Wananchi Group Stores Requisition System.

It closely follows the original "HOW TO USE" process while making it paperless (until physical issuance).

### Key Features Demonstrated
- Role-based navigation (Requester, Department Head, Store Team, Management)
- Colour-coded status system (Yellow → Green → Red → Blue → Grey)
- Auto-generated REF NO, auto-calculations, and auto-approver
- New Requisition Form
- Central Requisitions Log with powerful filters
- Reports & Dashboard
- Gate Pass print preview
- Responsive design
